Aplicação CRUD em PHP.

Instruções:
1. Instale o servidor web Apache e o bando de dados MySQL em sua máquina.
2. Baixe o repositório em seu computador.
3. Crie um banco de dados no MySQL com nome "crud-app".
4. Importe os arquivos juntamente com a pasta "desafiocrud" para "C:\xampp\htdocs"
5. Configure credencias de acesso ao banco de dados no arquivo "db_connect.php" dentro da pasta "desafiocrud\php_action"
6. Inicie o servidor web Apache
7. Abra o navegador e acesse o endereço http://localhost/desafiocrud

Funcionalidades:

-Cadastro de clientes;
-Edição de clientes;
-Exclusão de clientes;
-Listagem de todos os clientes cadastrados.

Estrutura do projeto segue o padrão MVC:

-Model: responsável pela interação com banco de dados
-View: contém páginas HTML da aplicação
-Controller: Recebe requisições do usuário e retornam as respostas.

Esta é uma aplicação básica em CRUD PHP, é possível aperfeiçoar e evoluir muito mais apartir desta aplicação.
